## 0. 通用设计前置（3 个方案共用）

### 0.1 iOS 26 Liquid Glass 关键要点（开发者视角）

iOS 26 引入的新一代材质系统，区别于 iOS 7-25 的 `Material.regularMaterial` 半透明模糊：

| 特性 | Liquid Glass | 旧 Material |
|---|---|---|
| 折射 | **有**（背景透过玻璃产生折射形变） | 无 |
| 边缘高光 | **有**（动态高光随视图运动） | 无 |
| 厚度 | 可感知"层级深度" | 平面 |
| 自适应 | 实时根据下层内容调整明度/色温 | 静态模糊 |
| API | `.glassEffect(_:in:)` / `GlassEffectContainer` | `.background(.regularMaterial)` |

**预设样式**：
- `.regular` — 标准玻璃（默认）
- `.clear` — 更透更薄
- `.tinted(Color)` — 带主题色染色

**关键约束**：
- Liquid Glass 应用在**悬浮元素**上（Tab Bar / 导航栏 / 浮窗 / 卡片），不应用在大面积底色
- 下层必须有彩色内容/壁纸，否则玻璃感等于消失
- 暗色模式下需调高 tint 透明度防止变成纯黑

### 0.2 三方案共用规则

- **底部 Tab Bar** 全部用 Liquid Glass `.regular`
- **顶部导航栏** 全部用 Liquid Glass `.regular`
- **挂牌卡（CertificateView）** 是 App 灵魂资产，3 个方案的卡片设计差异最大
- 所有按钮：胶囊形 + Liquid Glass tinted
- "仅供娱乐"标识统一为右上角金色小徽章
- 字体保持 PROJECT_BRIEF §6.3：标题 PingFang SC Semibold / 编号 SF Mono

### 0.3 评估维度（方案对比锚点）

| 维度 | 权重 | 说明 |
|---|---|---|
| 与小红书 70% 女性用户审美契合度 | ★★★★★ | 决定冷启动是否能爆款 |
| 与"鉴定/品相"调性契合度 | ★★★★ | 太可爱会失去"鉴定权威感"，太严肃失去萌点 |
| iOS 26 Liquid Glass 应用充分度 | ★★★★ | 占新 OS 红利、显得专业 |
| 实现难度 | ★★★ | 个人开发者 4-5 周内可完成 |
| 与 PROJECT_BRIEF §6 设计哲学兼容度 | ★★★ | 配色基本沿用、字体复用 |
| 男性/中性用户接受度 | ★★ | 加分项不是必需 |

### 1.6 Liquid Glass 关键应用

- **导航栏 / Tab Bar**：`.glassEffect(.tinted(朱砂橙).regular)` — 让传统色透过现代材质
- **鉴定卡片**：`.glassEffect(.regular, in: RoundedRectangle(cornerRadius: 16))` — 卡片悬浮在宣纸背景上
- **挂牌卡内的"印章圆"**：`.glassEffect(.clear)` + 朱砂叠加 — 印章有玻璃光泽
- **鉴定中浮层**：`GlassEffectContainer` 包裹照片框，下层朱砂光晕透出

    }

### 3.6 Liquid Glass 关键应用

- **导航栏 / Tab Bar**：`.glassEffect(.clear)`（最薄最透） + 樱花粉极淡染色
- **入口卡片**：纯白底 + Liquid Glass `.clear` 浮层效果 — 像奶油托盘上的小蛋糕
- **照片框**：`.glassEffect(.regular)` + 圆角 24-28pt — 照片像浮在玻璃水珠里
- **挂牌卡**：白底 + 樱花粉边框光晕 — 玻璃感不强，但温暖感强（这是有意的——挂牌卡是要发小红书的，需要清晰可读，不能太透）

## 5. 我的推荐 → 方案 C（奶油绒感）

### 5.1 推荐理由

1. **用户基本盘对齐**：SPEC §2.1 锁定 70% 女性 18-35 岁小红书用户，方案 C 的视觉语言和该群体审美完全对齐
2. **冷启动渠道适配**：PROJECT_BRIEF §9 的小红书冷启动策略，方案 C 的挂牌卡天然是"小红书素材"

4. **延续 BRIEF 设计哲学**：BRIEF §6 已定义的"Playful Luxury"在方案 C 中得到最忠实的延续
5. **反差萌靠文案救场**：方案 C 视觉偏可爱，但 SPEC §4.3 的"血统点评"文案完全可以制造反差权威感（"村霸气质 + 御猫风骨"打在奶油色证书上，反差感更强）

### 5.3 不推荐：方案 B
